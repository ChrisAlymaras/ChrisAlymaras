## C# Small Banking System — Admin / Customer Experience (Coming Soon!)
Currently developing a small Banking System where Administrators can execute all CRUD operations for users and accounts while Customers can make internal/external transfers from their bank account and request exchange rates to display balance in other currencies.
Features:
- 🖥️ MVC Architecture: Built with high-performance Controllers and Razor Pages, implementing clean code principles and common design patterns.
- 🔐 Authentication & Authorization: Secure user identity management using ASP.NET Core Identity with with role-based access control.
- 🗄️ SQL Server Integration: A relational database backend to reliably store user profiles, accounts, balances and conversion rates.
- 💸 Currency Conversion: Integration with fixer.io API to provide real-time exchange rates and display balances in multiple currencies.

## 🦸 React Pokedex - Interactive Pokemon Explorer 🔗 Live Demo [Pokedex App on Netlify](https://pokedexappchrisalym.netlify.app/)
Pokedex is a web application built with **React**, fetching data from the public **Pokedex API**. It uses dynamic interfaces and smooth UI transitions to render data in real time based on user's search terms and requests.
Features:
- 🐾 Fetch all pokemons via API and explore in a scrollable way 
- 🔍 Search for specific Pokemon by name and update instantly
- 📈 View detailed Pokemon stats through interactive and user-friendly tabs
- 💥 Request the evolution chain through images

## ⛅ React Weather App - Real Time Weather Dashboard 🔗 Live Demo [Weather App on Netlify](https://weatherappchrisalym.netlify.app/)
Weather App is the latest web project I built using **React** and the **OpenWeather API**. The app offers city / state search delivering current weather conditions, 5-day forecast, and favourites system using LocalStorage.
Features:
- 🎯 OpenWeather API integration for real-time weather data
- 📌 Caching layer to reduce API calls and ensure better performance 
- ❄️ Dynamic weather based backgrounds
- 📉 Scrollable weather cards for forecasting
- ❤️ Favourites section for quick accessibility
